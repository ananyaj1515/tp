### Hall Ledger

[![Java CI](https://github.com/AY2526S2-CS2103T-T15-1/tp/actions/workflows/gradle.yml/badge.svg)](https://github.com/AY2526S2-CS2103T-T15-1/tp/actions/workflows/gradle.yml)

![Ui](docs/images/Ui.png)

- This is **a sample project for Software Engineering (SE) students**.<br>
  Example usages:
  - as a starting point of a course project (as opposed to writing everything from scratch)
  - as a case study
- The project simulates an ongoing software project for a desktop application (called _Hall Ledger_) used for managing student records for those living in halls.
  - It is **written in OOP fashion**. It provides a **reasonably well-written** code base **bigger** (around 6 KLoC) than what students usually write in beginner-level SE modules, without being overwhelmingly big.
  - It comes with a **reasonable level of user and developer documentation**.
- It is named `Hall Ledger` (`HL` for short) because it manages student records for those living in halls and keeps track of the CCA points, merit/demerit points, room allocations, and other confidential student details.
- For the detailed documentation of this project, see the ![Hall Ledger User Guide](https://ay2526s2-cs2103t-t15-1.github.io/tp/)
